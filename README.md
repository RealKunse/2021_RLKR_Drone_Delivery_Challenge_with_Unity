# RLKorea Drone Delivery Challenge

## :fire: 개요

본 챌린지는 RLKorea에서 주최하는 강화학습 기반의 드론 물품 배송 챌린지입니다. 본 챌린지는 강화학습 알고리즘으로 드론을 학습하여 물품을 목표지점인 집으로 충돌없이 빠르게 배송하는 것을 목표로 합니다.
